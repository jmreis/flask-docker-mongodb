[Docker](https://docs.docker.com/) e banco de dados [Mongo DB](https://www.mongodb.com/docs/).



## Requisitos
----------

- docker
- docker-compose


## Instalação
----------

Use git:

```bash

    $ git clone https://github.com/jmreis/flask-docker-mongodb

    $ cd python-flask-rss-feeds

    $ docker-compose up --build

```
